¿Qué es Apache Flink?

⚡

Streaming Real-Time

No espera a que los datos se acumulen; los procesa en milisegundos según llegan al sistema.

🧠

Stateful (Estado)

Es capaz de "recordar" la trayectoria previa de un avión para compararla con su estado actual.

📈

Escalabilidad

Diseñado para manejar flujos masivos de datos a nivel mundial sin despeinarse.

Tú dices: "Elegimos Apache Flink porque, a diferencia de otros motores de datos, nos permite reaccionar al instante. No analizamos logs del pasado, analizamos eventos que están ocurriendo ahora mismo."

El Concepto del Proyecto

🧭 El control aéreo tradicional es excelente gestionando rutas individuales, pero falla al detectar patrones globales de forma automática y predictiva.

📡 Nosotros ingerimos datos públicos de OpenSky Network (señales ADS-B) para crear una capa geométrica automática que detecte riesgos sin que un humano tenga que estar mirando cada punto en el mapa.

¿Qué anomalías detectamos?

📉 Altitud Brusca

"Si un avión cae 5000 pies en 30 segundos, es una emergencia. Flink lo detecta y lanza la alerta inmediatamente."

🏢 Alta Densidad

"Si muchos aviones reportan anomalías en una misma celda H3, probablemente hay una tormenta severa o un fallo sistémico."

🗺️ Zonas Muertas

"Si en una ruta que siempre tiene tráfico de repente no hay nadie, algo está pasando. El sistema identifica estos 'vacíos' visualmente."

Arquitectura del Sistema

🌐
Ingesta: Capturamos datos continuos de la API de OpenSky.
📨
Mensajería: Un Poller en Python envía los datos a Kafka KRaft (nuestro sistema nervioso central).
🧠
Cerebro: Apache Flink aplica las reglas de detección sobre el flujo de Kafka.
🔌
Entrega: Los resultados salen servidos por una API REST/WebSocket en FastAPI.
🖥️
Visualización: El usuario final lo ve todo renderizado en un Dashboard Web interactivo.

¿En qué punto estamos?

✅ Contrato de Datos: Tenemos un JSON unificado cerrado y el diseño visual mockeado en el Dashboard.

✅ Infraestructura: La red Docker ya está levantada de forma autónoma con Kafka KRaft y Flink operativos.

🚀 En curso: Estamos en pleno desarrollo de los 'Jobs' de procesamiento (la lógica del cerebro de Flink), que es el siguiente gran Hito.
