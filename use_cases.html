UC-01

Alerta Vertical — Ascenso / Descenso Brusco

Detecta cuando una aeronave experimenta un cambio de altitud anómalo en un periodo corto de tiempo. Flink mantiene un ValueState con el historial reciente de altitudes por cada icao24. Si la diferencia entre la altitud actual y la previ (o el vertical_rate reportado por OpenSky) supera el umbral, se genera una alerta. Utiliza el campo de estado del ciclo de vida para no inundar el frontend.

keyBy(icao24) ValueState ProcessFunction flight-alerts

sequenceDiagram
    participant K as 📨 Kafka raw-flights
    participant F as ⚡ Flink ProcessFunction
    participant S as 💾 ValueState
    participant A as 🚨 Kafka flight-alerts

    K->>F: FlightState (icao24=3c6752, alt=11200)
    F->>S: Guardar alt = 11200
    K->>F: FlightState (icao24=3c6752, alt=9100)
    F->>S: Comparar con historial (Δ= -2100m)
    S-->>F: Actualizar a 9100, marcar alerta ACTIVA
    F->>A: ⚠️ ALERTA VERTICAL [estado: inicio]

UC-02

Alta Densidad — Concentración de Aeronaves (H3)

Agrupa los eventos por su celda geográfica H3 dentro de una ventana tumbling de 5 minutos, lo que permite funcionar sin depender de bases de datos externas de aeropuertos. Si en una celda espacial se acumulan más de X aeronaves, Flink emite una alerta de alta_densidad apuntando a las coordenadas centrales de esa celda H3.

Índice H3 keyBy(celdaH3) Tumbling Window 5min AggregateFunction

UC-03

Alerta de Zona Muerta — Caída Abrupta de Tráfico

Compara el tráfico de cada celda hexagonal H3 contra la ventana inmediatamente anterior mantenida en el estado de Flink. Si una celda que tenía una cantidad significativa de actividad (ej: 12 vuelos) pasa repentinamente a tener 0 vuelos o una caída crítica, se genera una alerta de zona muerta. Esto evita la necesidad de cargar históricos complejos.

Índice H3 keyBy(celdaH3) Tumbling Window (Previous State)

UC-04

Flink CEP — Detección de Patrones de Emergencia

Utiliza la API de Flink CEP (Complex Event Processing) para definir secuencias de condiciones estrictamente basadas en los datos de /states/all de OpenSky.

Patrón: Un avión ejecuta un descenso brusco seguido inmediatamente de que el campo on_ground = true y velocity = 0 durante más de 60 segundos → posible aterrizaje forzoso. Al recuperarse la altitud estable u otra condición, enviará estado: "fin".

Flink CEP Pattern API within(2 min) 
%%{init: {'theme': 'dark', 'themeVariables': {'fontSize': '12px', 'fontFamily': 'Inter'}}}%%
stateDiagram-v2
    direction LR

    [*] --> Monitorizando
    Monitorizando --> Descenso: Δ alt > 2000m
    Descenso --> OnGround: on_ground = true
    OnGround --> AlertaCritica: vel = 0 (durante 60s)
    AlertaCritica --> [*]: Emitir alerta [estado: inicio]
